# meme_xd

A MemeApp with 5 screens 
1. Login 
2. Albums
3. See Albums
4. To Do List 
5. Users Profile

## Getting Started
To run the application run the attached apk file or 
1.create a new flutter project in android studio 
2.replace the lib and pubspec.yaml file
3.build the project in a emulator
